# JMeter Scripts

This folder contains Apache JMeter test scripts for API Performance Testing.
API Login Performance Test
Project Overview
This project demonstrates Performance Testing of a Login REST API using Apache JMeter.

Objective
Validate the Login API performance under expected user load and identify any performance bottlenecks.

Tools Used
Apache JMeter
CSV Data Set Config
HTTP Request Sampler
HTML Dashboard
Test Scenario
Login API
100 Virtual Users
Ramp-Up: 5 Minutes
Duration: 20 Minutes
KPIs
Average Response Time
Throughput
Transactions Per Second (TPS)
Error Percentage
Status
This project is under development. More scripts, reports, and screenshots will be added.
